# Splunk Push

Ruby gem to use splunk HTTP Event Collector.

## Usage

In Gemfile add:

```
gem 'splunk_push', github: 'korki/splunk_push'
```

Set environmental variables:

```
ENV['SPLUNK_TOKEN']

ENV['SPLUNK_COLLECTOR_URL']
```
